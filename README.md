# lifex-landing

Public marketing site for LIFEX OS — `lifexos.co.in`.

## Stack

React 18 · TypeScript · Vite 5 · framer-motion · lucide-react

## Develop

```
npm install
npm run dev
```

Opens at `http://localhost:5173`.

## Build

```
npm run build
```

Output in `dist/`.

## Scope

Phase 1 of LIFEX commercial transformation — see `../LIFEX_COMMERCIAL_ARCHITECTURE.md`.
